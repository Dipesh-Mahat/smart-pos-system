How do I add new inventory items?
To add new inventory items:
  1. Navigate to the Inventory section from the main menu
  2. Click the "Add New Item" button
  3. Fill in the required details: item name, SKU, price, quantity, category
  4. Upload product images if available
  5. Set minimum stock alerts and reorder levels
  6. Click "Save" to add the item to your inventory
You can also bulk import items using Excel/CSV files from the Import option.
How can I process a refund or return?
To process a refund:
  1. Go to Transactions and find the original sale
  2. Click on the transaction to view details
  3. Select "Process Refund"
  4. Choose items to refund (full or partial)
  5. Select refund method (cash, card, store credit)
  6. Add refund reason for record keeping
  7. Print refund receipt for customer
Note: Refunds will automatically update inventory levels.
How do I generate sales reports?
To generate sales reports:
  1. Navigate to Reports section
  2. Select report type: Daily, Weekly, Monthly, or Custom
  3. Choose date range for the report
  4. Select specific filters (products, categories, staff)
  5. Click "Generate Report"
  6. View, print, or export the report as PDF/Excel
Reports include sales summary, top products, profit margins, and payment methods breakdown.
How to manage user permissions and roles?
To manage user permissions:
  1. Go to Settings → User Management
  2. Select an existing user or add a new one
  3. Choose from predefined roles: Admin, Manager, Cashier, or Custom
  4. Set specific permissions for each module
  5. Configure access hours and restrictions
  6. Save changes and notify the user
Roles determine what features users can access and modify in the system.
How to set up tax rates and configurations?
To configure tax settings:
  1. Navigate to Settings → Tax Configuration
  2. Add new tax rate (VAT, Service Tax, etc.)
  3. Enter tax percentage and description
  4. Assign taxes to specific product categories
  5. Set default tax behavior for new products
  6. Configure tax exemptions if applicable
The system supports multiple tax types and automatic calculation during sales.
How to connect receipt printers and hardware?
To connect hardware:
  1. Connect your device (printer, scanner, cash drawer) via USB or network
  2. Go to Settings → Hardware Devices
  3. Click "Add New Device"
  4. Select device type and connection method
  5. Test the connection to ensure proper setup
  6. Configure device-specific settings
Supported devices include thermal printers, barcode scanners, and cash drawers.
How to backup and restore data?
To backup your data:
  1. Go to Settings → Backup & Restore
  2. Choose backup type: Full system or Selective
  3. Select backup location (local drive, cloud, network)
  4. Set automatic backup schedule
  5. Initiate backup process
  6. Verify backup completion and integrity
Regular backups ensure your business data is always protected and recoverable.
How to manage multiple store locations?
To manage multiple stores:
  1. Navigate to Settings → Store Management
  2. Add new store location with details
  3. Configure store-specific settings and tax rates
  4. Assign staff members to specific locations
  5. Set up inventory transfer between stores
  6. Generate location-specific reports
The system supports centralized management with location-specific customizations.
